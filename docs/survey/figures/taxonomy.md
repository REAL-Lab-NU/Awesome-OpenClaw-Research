## Classification Criteria

| Layer | Question It Answers | Criterion |
|-------|-------------------|-----------|
| L1 | How is the framework extended? | Modifies or extends the OpenClaw platform itself |
| L2 | How secure/efficient is the marketplace? | Analyzes or improves the skill ecosystem |
| L3 | What are the threats and defenses? | Security analysis, attacks, or mitigations |
| L4 | How do agents behave socially? | Studies agent behavior on Moltbook or similar platforms |
| L5 | Where is it being applied? | Uses OpenClaw in a specific domain |
| L6 | What does the ecosystem mean? | Meta-analysis, surveys, strategic assessment |
